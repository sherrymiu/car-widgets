# 汽车智能座舱年轻化 - 交互Widget展示

## 项目介绍

这是一个汽车智能座舱年轻化概念设计的交互展示项目，包含两个创意Widget：

### 🐠 Fish Tank Widget
动态生态系统交互设计，展现年轻、灵动的视觉效果。

### 🎛️ Vinyl DJ Widget  
音乐交互体验设计，融合潮流审美与功能交互。

## 快速开始

### 在浏览器中打开
直接打开 `index.html` 即可查看所有Widget，支持：
- 🖥️ 桌面浏览器（Chrome、Safari、Firefox 等）
- 📱 移动设备（iOS Safari、Android Chrome 等）
- 📊 平板设备（完全响应式适配）

### 本地运行
推荐使用本地服务器以获得最佳体验：

```bash
# 使用 Python
python -m http.server 8000

# 或使用 Node.js http-server
npx http-server

# 或使用 VS Code Live Server 扩展
```

然后访问 `http://localhost:8000`
